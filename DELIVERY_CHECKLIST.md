## 🏗️ 架构亮点

### 前后端分离
```
前端: Vue 3 SPA
    ↓ HTTP/JWT
后端: Express API
    ↓ SQL
数据库: SQLite
```

### 模块化设计
- 控制器层：业务逻辑
- 路由层：API 端点
- 中间件层：请求处理
- 工具层：通用功能
- 状态层：前端状态管理

### API 设计
- 13 个 RESTful 端点
- 标准 HTTP 方法
- JWT 认证
- JSON 数据格式

## 💾 数据库设计

### 三张表结构
```
users (用户表)
├── id: 主键
├── username: 用户名（唯一）
├── email: 邮箱（唯一）
├── password: 加密密码
└── created_at: 创建时间

categories (分类表)
├── id: 主键
├── user_id: 用户 ID（外键）
├── name: 分类名
├── color: 分类颜色
└── created_at: 创建时间

expenses (花销表)
├── id: 主键
├── user_id: 用户 ID（外键）
├── category_id: 分类 ID（外键）
├── amount: 金额
├── description: 描述
├── date: 日期
└── created_at: 创建时间
```

### 关系图
```
users (1:N) categories (1:N) expenses
```
